## Built With

### Backend

- [Laravel](https://laravel.com/) - The PHP Framework for Web Artisans.
- [MySQL](https://www.mysql.com/) - Local mysql database storage.
- [Neon](https://neon.com/) - Online pgsql database storage.
- [Guzzle](http://docs.guzzlephp.org/en/stable/) - PHP HTTP client for API requests.

### Frontend

- [Laravel Blade](https://laravel.com/docs/blade) - Templating engine.
- [Tailwind](https://tailwindcss.com/) - Styling.

### API End Point

- [OMDb API](http://www.omdbapi.com/) - Movie database source.

## Usage

```
php artisan migrate
php artisan db:seed
php artisan serve
```
